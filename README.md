# foresty

## Introduction

Foresty는 개인 메모 작성 및 질문과 답변이 가능한 웹 애플리케이션입니다.
Foresty의 백엔드 깃입니다.

#개인 메모에 질문 및 댓글 기능은 현재 중지되었습니다.

### Stack
Node.js(Javascript), Express, mongoDB, mongoose, Firebase, AWS EC2
